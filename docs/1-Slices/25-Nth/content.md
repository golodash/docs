---
Title: Nth
---


Gets the (n)th element of a slice. If n is negative, the (nth)
element from the end is returned

#- Inputs
- slice `[]anything`
- index `int`


**Note**: Passing invalid input causes panic.

#- Outputs
- first `sameTypeAsSliceElement`

#- Complexity
- `Nth` has a complexity of O(1).
